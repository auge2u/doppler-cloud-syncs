### Added
- Initial project structure with pnpm monorepo
- `@auge2u/dcs` CLI with commands: init, sync, status, diff, run
- `@auge2u/dcs-runtime` with Firebase, Cloudflare, and Cloud Run adapters
- `@auge2u/dcs-dotfiles` for shell integrations
- Neon database branch management commands
- Git hooks for automatic secret syncing
- Webhook support for Doppler change notifications
- Comprehensive test suite (181 tests)
- GitHub Actions CI/CD workflows
- Changesets for version management
